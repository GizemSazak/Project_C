Django openen
Om django database te openen eerst moet je naar de file waarin staat manage.py en daarna
moet je shift ingedrukt houden en op rechter muisknop klikken, vervolgens klik je op powershell-venster openen 
en daarna kan je dit command gebruiken.
py manage.py runserver 
Dan krijg je de linke server http://127.0.0.1:8000/.
Je moet naar admin panel gaan dus je schrijft http://127.0.0.1:8000/admin, daarna vraagt hij je 
om de admin naam en admin wachtwoord in te voeren dus je vult in "admin" als gebruiksnaam en wachtwoord.
Als je iets wil veranderen bij database dan moet je die commdends line gebruiken.
py manage.py makemigrations
py manage.py migrate
Als je een nieuw app voor de database wil maken dan kan je de command 
py manage.py startapp mijnapp (inplaats van mijnapp zet je je app naam)
